# ReliabilityOps Platform

A production-style SRE and DevOps portfolio project.

This project will start with a small backend API and gradually grow into a complete reliability platform with:

- Go backend API
- PostgreSQL database
- Docker
- Kubernetes
- CI/CD with GitHub Actions
- GitOps with ArgoCD
- Monitoring with Prometheus
- Dashboards with Grafana
- Alerting with Alertmanager
- Incident simulation
- SRE runbooks
- AI-assisted incident summaries
